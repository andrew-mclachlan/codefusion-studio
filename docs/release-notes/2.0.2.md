## About this release

CodeFusion Studio 2.0.2 is a patch release that fixes a regression affecting CFS [Telemetry](../user-guide/about/telemetry.md).

In CodeFusion Studio 2.0.1, telemetry events were not collected due to this regression. This impacted the following event types:

- Workspace creation events
- Extension activation events

Ensure that you are running the CodeFusion Studio 2.0.2 VS Code extension to resolve this issue.

If extension auto-updates are enabled, no action is required. For installation or upgrade steps, refer to [Install the CodeFusion Studio VS Code extension](../user-guide/installation/install-extensions.md).

!!! note
    CodeFusion Studio 2.0.2 is distributed only through the Visual Studio Code Marketplace. A standalone installer is not provided for this release, so manual installation using a VSIX file is not supported.
